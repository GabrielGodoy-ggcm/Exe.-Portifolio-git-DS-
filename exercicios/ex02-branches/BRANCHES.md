## Comandos principais usados

| Comando | O que faz |
|---|---|
| `git checkout -b nome-da-branch` | Cria e entra na branch |
| `git push origin nome-da-branch` | Envia branch para o GitHub |
| `git checkout main` | Volta para a main |
| `git merge nome-da-branch` | Mergea a branch na main |
